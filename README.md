# Eureka Cannabis Brand Website

A minimal, modern, and high-performance landing page framework built for Eureka.

## Features
- **Age-Verification Modal:** Prevents underage access by checking local browser storage before allowing viewing.
- **Tailwind CSS Styling:** Uses clean design tokens, warm off-white (`stone-50`) backgrounds, and subtle `emerald-600` accents.
- **Responsive Layout:** Automatically scales cleanly from mobile displays to large desktop monitors.

## Quick Start
1. Clone this repository or download the files.
2. Open `index.html` directly in any web browser, or launch it with a local server package like Live Server on VS Code.
3. Replace placeholder background containers inside the code with your specific product photos and hero media.

## Folder Structure
```
eureka-website/
├── index.html          # Main website and layout
├── README.md           # Instructions on how to run/deploy
└── assets/             # Folder for logos and product photos
```

## Live Demo
Your website is live at: **https://hugoboss201.github.io/eureka-website/**

## Customization
- Update product images in `assets/` folder
- Replace placeholder text with your actual product details
- Customize colors by modifying Tailwind classes (currently using `emerald-600` for accents)
- Add your actual license number in the footer
