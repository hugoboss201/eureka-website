# TopShelfEureka.com

Modern cannabis brand website for Eureka.

## Run locally
```bash
npm install
npm run dev
```

## Build for hosting
```bash
npm run build
```

Upload to Vercel, Netlify, or any static host that supports Vite React.

## Included pages/sections
- Hero homepage
- Iconic Drops section using your provided image
- Product/drop cards
- Menu feature section
- Wholesale inquiry CTA
- About section
- Mobile responsive navigation
